# SentimentAnalysis
A simple machine learning project that classifies movie reviews as positive or negative using logistic regression. Built with Python and Scikit-learn, this project demonstrates the basics of data preprocessing, model training, and evaluation.

A beginner-friendly machine learning project that classifies movie reviews as **positive** or **negative** using logistic regression.  
This project demonstrates the basics of **data preprocessing, model training, evaluation, and visualization** with Python and Scikit-learn.
